How It Works

  1. Scan the QR code: The QR code always points to the /qr endpoint.
  2. Dynamic redirect: When scanned, the QR code redirects users to the URL currently set in the system.
  3. Update anytime: Use the form above to update the redirect URL dynamically without changing the QR code.
  4. Instant effect: Changes take effect immediately, so all scans after update redirect to the new URL.
  5. Easy management: This allows flexible redirection for marketing campaigns, events, or any use case.
